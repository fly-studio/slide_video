# Slider - 幻灯片特效视频生成器

## 项目概述
使用OpenCV生成视频帧，通过ffmpeg管道生成H.264格式视频的幻灯片特效系统。

## 核心功能
- **帧生成**：使用OpenCV生成每一帧
- **视频输出**：通过ffmpeg管道写入，生成.264格式视频
- **可配置参数**：FPS、输出分辨率

## 技术栈
- **Python**: 3.12.11
- **包管理器**: uv 0.8.20
- **核心依赖**:
  - opencv-python 4.12.0.88 (cv2) - 图像处理和帧生成
  - numpy 2.2.6 - 数组计算
  - ffmpeg - 视频编码（外部依赖）

## 项目结构
```
slider/
├── main.py          # 主入口（待实现）
├── sideshow.py      # 数据模型定义
├── effect.py        # 特效实现（待实现）
├── 0-20.jpg         # 输入图片（21张）
└── .venv/           # 虚拟环境
```

## 数据模型
- **SlideEffect**: 特效定义（duration, effect类型, expr表达式）
- **Slide**: 单张幻灯片（file_path, in_effect, hold_effect, out_effect）
- **Sideshow**: 幻灯片集合

## 视频规格
- **输入图片**: 1152x2048 像素
- **输出视频**: 720x1280 像素（可配置）
- **FPS**: 可配置
- **每张幻灯片时长**: 4.5秒
  - 入场特效: 0.5秒
  - Hold效果: 3.5秒
  - 出场特效: 0.5秒
